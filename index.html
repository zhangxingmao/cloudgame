Om Nom Run

Join the adorable green monster Om Nom in this exciting endless runner where you'll dodge obstacles and collect sweet treats!

Game Description

Om Nom Run is an exciting endless runner game featuring the lovable green monster from the Cut the Rope series. Guide Om Nom through a colorful world filled with obstacles and challenges while collecting candies and power-ups along the way.

With simple controls but challenging gameplay, Om Nom Run offers hours of entertainment for players of all ages. The further you run, the more difficult the obstacles become - how far can you go?

How to Play

  • • Use the arrow keys or swipe to move Om Nom left and right
  • • Jump over obstacles by pressing the up arrow or swiping up
  • • Slide under barriers by pressing the down arrow or swiping down
  • • Collect candies to increase your score
  • • Grab power-ups for special abilities
  • • Try to beat your high score with each run!

About Om Nom Run

Om Nom Run is developed by ZeptoLab, the creators of the popular Cut the Rope series. Featuring the same charming character and colorful graphics that fans love, this endless runner brings a new dimension to the Om Nom adventure.

The game features smooth animations, responsive controls, and an increasing difficulty curve that keeps players coming back for more. Whether you're a longtime fan of Om Nom or new to the character, this addictive runner game offers fun gameplay that's easy to learn but challenging to master.
